# Mosaiq Chromium Fork — Patch Series（quilt 风格）
#
# 每行一个 patch 文件名（相对于 patches/ 目录）。
# 顺序即应用顺序 —— scripts/apply-patches.sh 按这里的顺序 git apply。
#
# 命名约定：NNNN-kebab-case-name.patch
#   NNNN = 4 位序号
#     0001-0010 = 基础反检测 patches（Canvas / WebGL / Audio / UA / UA-CH / Screen / Hardware / Timezone / Fonts / WebRTC）

#     0015      = WebUI 注册（chrome://mosaiq/*）
#

#   3. 0011-tls-ja4-spoof        ← 最难，BoringSSL + QUIC 同步
#
# Phase A.0 状态：series.txt 为空。三个 patch 的设计稿在 patches/*.spec.md
