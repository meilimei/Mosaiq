Captured JSON

What gets captured? 
- UNMASKED_VENDOR_WEBGL / UNMASKED_RENDERER_WEBGL via WEBGL_debug_renderer_info
- ~30 WebGL1 capability params (MAX_TEXTURE_SIZE, MAX_VIEWPORT_DIMS, ...)
- ~25 WebGL2 capability params (MAX_3D_TEXTURE_SIZE, MAX_SAMPLES, ...)
- VERSION / SHADING_LANGUAGE_VERSION (WebGL1 + WebGL2 strings)

Not captured:
- Any context-dependent state (current buffer / texture / framebuffer)
- Canvas pixel content
- Screen / window / cookies / storage
